프로그램 설명
0.메인 메뉴.
	화살표 위 아래로 드롭다운 메뉴를 움직일 수 있습니다.
오른쪽 화살표를 눌러 원하시는 메뉴에 진입할 수 있습니다.
1.Search
	세 가지 옵션(Name,Phone,Memo)로 검색할 수 있습니다.
각 메뉴는 오른쪽 화살표를 통해 진입할 수 있습니다.
각 메뉴에 진입 후에는 검색하고자 하는 내용을 입력 후 Enter를 눌러 검색을 시작합니다.
검색에서 원하는 결과가 있다면 해당 결과의 Name, Phone, Memo를 보여줍니다.
검색에서 원하는 결과가 없다면 There is no data를 반환합니다.
검색이 끝났다면 Y나 N을 눌러 프로그램을 종료하거나, 다시 메인 메뉴로 나갈 수 있습니다.(Enter를 누를 필요가 없습니다.)

2.Add
	추가하고자 하는 데이터를 입력받습니다.
설명에 따라 Name , Phone , Memo를 입력받습니다.
각각의 데이터는 입력후 Enter를 눌러 입력을 완료합니다.
추가가 성공적으로 진행되었다면 Add Success!를 반환합니다.
추가가 성공적이지 않았다면 Add Faile.을 반환합니다.
추가가 끝난 후에는 Y나 N을 입력하여 프로그램을 종료하거나 다시 메인 메뉴로 나갈 수 있습니다.(Enter 를 누를 필요가 없습니다.)
3.Delete
	구현하지 못했습니다. 죄송합니다.
4.List
	옵션에 진입 시 전화번호부의 데이터를 출력해 줍니다.
해당 옵션은 y픽셀의 크기가 35픽셀 이상일 경우에만 정상적으로 보입니다. 주의해 주세요.
데이터 출력이 끝난 후에는 Y나 N을 눌러 프로그램을 종료하거나 다시 메인 메뉴로 나갈 수 있습니다.(Enter를 누를 필요가 없습니다.)

5.Exit
	프로그램을 종료합니다.

데이터 구조 설명.
	구조체 배열을 선언 후 해당 구조체 배열을 각 기능을 하는 함수에 인자로 전달하여 처리했습니다.

컴파일 방법.
gcc -o tel tel.c -lncurses 입니다.
